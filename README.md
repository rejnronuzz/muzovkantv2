# muzovkantV2
### shitty (half-)vibecoded discord bot
## overall
you can use this as a base for your discord bot, writing your own [cogs](/cogs/), or as is.


## installation
firstly, clone the repo
```
git clone https://github.com/rejnronuzz/muzovkantv2.git
cd muzovkantv2
```
then insert your discord bot and your [thecatapi](https://thecatapi.com) token into the .env file.
it should look something like this:
```
DISCORD_TOKEN=abc123abc
CAT_API_KEY=live_abc123abc
```
you can then edit the config.py to your liking.
